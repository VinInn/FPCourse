# FPCourse
A Course on "Floating Point Arithmetic for scientists"

Floating point arithmetic is at the hearth of any scientific computation.
In this course I will introduce the basic concepts of floating point representation and arithmetic and discuss how precision can be preserved or disrupted.
I will cover the essentials of elementary function evaluation and conclude discussing how to optimize computations without compromising the required accuracy.
Practical examples and exercises will be proposed in both Python and C++.
